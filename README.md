# Traffic Accident Analysis

## Project Overview

This project analyzes traffic accident data and predicts accident severity using Machine Learning.

The dataset was cleaned and preprocessed using Python before training a Random Forest classification model.

## Objectives

- Clean traffic accident data
- Perform data preprocessing
- Explore accident trends
- Train a Random Forest model
- Predict accident severity
- Evaluate model performance

## Machine Learning Model

- Random Forest Classifier
